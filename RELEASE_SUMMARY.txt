================================================================================
                              QUICK START GUIDE
================================================================================

1. Install the package:
   sudo dpkg -i telegramdigger_0.5.0_amd64.deb

2. Set your bot token:
   export TGDIGGER_TOKEN="123456789:ABCdefGHI..."

3. Validate the token:
   telegramdigger --validate

4. Check admin rights:
   telegramdigger --read-botrights

5. View webhook status:
   telegramdigger --webhook-get

6. Get help:
   telegramdigger --help

================================================================================
                           COMMAND REFERENCE
================================================================================

Token Management:
  --validate                    Validate bot token via API
  --bulk-validate [FILE]        Validate multiple tokens from file

Bot Information:
  --read-botrights              Read default administrator rights

Webhook Operations:
  --webhook-get                 Get current webhook information
  --webhook-info                Get webhook status (alias)
  --webhook-set <URL>           Set webhook URL
  --webhook-delete              Delete webhook configuration

General:
  --help                        Show help message
  --version                     Show version information
  --about                       Show about information
  --token <TOKEN>               Specify bot token

Token Input Methods:
  1. CLI: --token <TOKEN>
  2. ENV: export TGDIGGER_TOKEN=<TOKEN>
  3. CFG: ~/.telegramdigger/settings.conf
